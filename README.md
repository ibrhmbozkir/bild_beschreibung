# Bildbeschreibung Training – Ibrahim

Kleine statische Website für Bildbeschreibung-Übungen (B1) mit klickbaren Punkten auf den Bildern.
- Startseite: `index.html`
- Detailseiten: `bild-33.html`, `bild-36.html`
- Bilder liegen im Ordner `bilder/` (eigene Fotos hier einfügen).

## Lokale Nutzung

Einfach `index.html` im Browser öffnen.

## GitHub Pages

1. Repository in GitHub anlegen und alle Dateien hochladen.
2. In den Repository-Einstellungen GitHub Pages aktivieren (Branch: `main`, Ordner: `/root`).
3. Die Seite ist dann unter der von GitHub angegebenen URL erreichbar.
